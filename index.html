RecordAR

Politica de Privacidad

Ultima actualizacion: 2026-03-12

Esta politica explica que datos recopila RecordAR, como se usan y que opciones tiene el usuario al utilizar la aplicacion.

1. Introduccion

RecordAR es una aplicacion movil que permite crear recuerdos multimedia vinculados a codigos QR. Esta politica explica que datos se recopilan, como se usan y que opciones tiene el usuario.

2. Datos que recopilamos

RecordAR puede recopilar y procesar los siguientes datos:

  • Datos de autenticacion proporcionados por Google, como nombre, correo electronico e identificador de usuario.
  • Contenido creado por el usuario dentro de la app, como titulo, descripcion y archivos o enlaces asociados a un recuerdo.
  • Identificadores tecnicos necesarios para el funcionamiento del servicio.
  • Contadores de escaneo asociados a cada recuerdo.

3. Como usamos la informacion

Usamos la informacion para:

  • permitir el inicio de sesion y la identificacion del usuario
  • guardar y mostrar recuerdos creados por el usuario
  • asociar recuerdos a codigos QR
  • mostrar un recuerdo cuando un QR es escaneado
  • contar la cantidad de escaneos de cada recuerdo
  • mantener la seguridad y operacion basica del servicio

4. Almacenamiento y proveedores

RecordAR utiliza servicios de Firebase, proporcionados por Google, para operar funciones como autenticacion, base de datos, almacenamiento y servicios asociados.

Dependiendo del uso de la aplicacion, los datos pueden almacenarse en:

  • Firebase Authentication
  • Cloud Firestore
  • Firebase Storage
  • Cloud Functions

5. Visibilidad del contenido

En la version actual del servicio, los recuerdos asociados a un codigo QR pueden ser visibles para usuarios autenticados que escaneen ese codigo.

Esto significa que el contenido no funciona como informacion estrictamente privada por propietario en todos los casos.

6. Permisos del dispositivo

La aplicacion puede solicitar permisos del dispositivo para:

  • usar la camara para escanear codigos QR
  • acceder a imagenes o archivos del dispositivo para crear recuerdos

Estos permisos se usan solo para las funciones solicitadas por el usuario.

7. Comparticion de datos

No vendemos informacion personal.

Los datos se comparten unicamente con proveedores de infraestructura necesarios para operar la aplicacion, principalmente Google Firebase, o cuando exista obligacion legal.

8. Conservacion de datos

Los datos se conservan mientras sean necesarios para operar la cuenta y los recuerdos del usuario, salvo que el usuario solicite su eliminacion o la ley exija otra cosa.

9. Seguridad

Aplicamos reglas de acceso y controles tecnicos sobre los datos almacenados. Sin embargo, ningun sistema es completamente infalible y no se puede garantizar seguridad absoluta.

10. Derechos del usuario

El usuario puede solicitar informacion o gestion relacionada con sus datos personales escribiendo al correo de soporte indicado al final de esta pagina.

11. Menores de edad

RecordAR no esta dirigida especificamente a menores de edad. Si detectamos uso indebido en contra de la normativa aplicable, podremos tomar medidas razonables.

12. Cambios a esta politica

Esta politica puede actualizarse en cualquier momento. La version vigente sera la publicada en la URL oficial de la politica de privacidad.
